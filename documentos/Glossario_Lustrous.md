# Glossário Técnico — Lustrous

Definições de termos técnicos utilizados na documentação do projeto.

---

## Perspectiva

| Termo | Definição |
|-------|-----------|
| **Vanishing Point** | Ponto de fuga — ponto no horizonte onde linhas paralelas parecem convergir |
| **Horizon Line** | Linha do horizonte — linha horizontal que representa o nível dos olhos do observador |
| **Eye-Level** | Nível dos olhos — altura vertical do ponto de vista do observador em relação à cena |
| **One-Point Perspective** | Perspectiva com 1 ponto de fuga — usada para cenas frontais |
| **Two-Point Perspective** | Perspectiva com 2 pontos de fuga — usada para cenas em ângulo |
| **Three-Point Perspective** | Perspectiva com 3 pontos de fuga — adiciona sensação de altura/profundidade extrema |
| **Grid Density** | Densidade do grid — quantidade de linhas de perspectiva exibidas (low/medium/high) |

---

## Visualização 3D

| Termo | Definição |
|-------|-----------|
| **OBJ** | Formato de arquivo 3D padrão contendo vértices, faces e normais |
| **Mesh** | Malha — estrutura de vértices e faces que forma a superfície de um objeto 3D |
| **Vertices** | Vértices — pontos no espaço 3D que definem os cantos da malha |
| **Faces** | Faces — polígonos (geralmente triângulos) formados por vértices que criam superfícies |
| **Trackball** | Controle de rotação esférica — permite girar objeto arrastando o mouse |
| **WebGL** | API JavaScript para renderização 3D acelerada por GPU no navegador |
| **Three.js** | Biblioteca JavaScript para criação e renderização de cenas 3D usando WebGL |

## Teoria das Cores

| Termo | Definição |
|-------|-----------|
| **RGB** | Modelo de cor aditivo: Red, Green, Blue (0-255 cada) |
| **HSV/HSB** | Modelo de cor: Hue (matiz), Saturation (saturação), Value/Brightness (brilho) |
| **HEX** | Notação hexadecimal para cores (#RRGGBB) |
| **Palette/Paleta** | Conjunto de cores utilizadas em uma imagem ou design |
| **Color Clustering** | Agrupamento de cores similares usando algoritmos como K-Means |
| **K-Means** | Algoritmo de clusterização que agrupa dados em K grupos por similaridade |
| **Dominant Colors** | Cores dominantes — cores mais frequentes ou visualmente impactantes em uma imagem |
| **Downsampling** | Redução de resolução para processamento mais rápido |
| **Temperature** | Temperatura de cor — classificação de cores como quentes (vermelho/amarelo) ou frias (azul/verde) |

## Interface & UX

| Termo | Definição |
|-------|-----------|
| **Burger Menu** | Menu hambúrguer — ícone de três linhas horizontais que expande menu de navegação |
| **Glassmorphism** | Estilo de design com fundo translúcido e efeito de vidro fosco |
| **Microinteractions** | Pequenas animações de feedback para ações do usuário |
| **Viewport** | Área visível do navegador onde o conteúdo é renderizado |
| **Client-side** | Processamento executado no navegador do usuário |
| **Export** | Funcionalidade de salvar resultado como arquivo (imagem, PDF) |

---

## Arquitetura

| Termo | Definição |
|-------|-----------|
| **Tool Controller** | Componente que gerencia estado e coordena inputs de cada ferramenta |
| **Domain Engine** | Módulo responsável pela lógica de negócio e cálculos específicos |
| **Renderer** | Componente responsável por desenhar/exibir o resultado visual |
| **Canvas 2D** | API do HTML5 para desenho de gráficos 2D |
| **Serverless** | Arquitetura onde código backend executa sob demanda sem servidor dedicado |
